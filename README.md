## Documentation Contributing Guidelines
Documentation is treated as code. 
- **Code:** All public classes and methods must include Javadoc.
- **Living Docs:** Complex algorithms must include a `<pre>` block with a JUnit test example.
- **Languages:** Please update both `README.md` and `README-uk.md`.
