## 🚀 Інструкція для нових розробників (Onboarding Guide)

Ласкаво просимо до команди розробки **Corgy Inclusive Game**! Цей посібник допоможе вам розгорнути проєкт на свіжій операційній системі (Windows/macOS/Linux).

### 1. Необхідне програмне забезпечення
Оскільки це нативний Android-застосунок на Java, вам знадобляться наступні інструменти:
* **Git:** для контролю версій ([Завантажити Git](https://git-scm.com/)).
* **Java Development Kit (JDK):** Версія 17 або вище (рекомендується Eclipse Temurin).
* **Android Studio:** Офіційне IDE для розробки ([Завантажити Android Studio](https://developer.android.com/studio)). Включає в себе Android SDK та емулятори.

### 2. Налаштування середовища розробки
1. Встановіть Git та Android Studio, використовуючи стандартні налаштування інсталяторів.
2. Відкрийте Android Studio та перейдіть у **SDK Manager** (Tools -> SDK Manager).
3. Переконайтеся, що у вас встановлено **Android SDK Platform 34** (або новішу версію) та **Android SDK Build-Tools**.

### 3. Клонування репозиторію та конфігурація
Відкрийте термінал (або Git Bash) та виконайте наступні команди:

```bash
# Клонування репозиторію
git clone [https://github.com/lizashalomeeva/Corgy_Inclusive_Game.git](https://github.com/lizashalomeeva/Corgy_Inclusive_Game.git)

# Перехід у папку проєкту
cd Corgy_Inclusive_Game
