Інформаційна система мобільного ігрового додатку з механізмами інклюзивної взаємодії користувача

Bachelor Thesis: Inclusive Android Gaming

Створення мобільної гри для Android, що адаптується під потреби людей з порушеннями зору та моторики за допомогою тактильного відгуку та високого контрасту.

Код на GitHub

Актуальність теми

У сучасному світі мобільні ігри часто ігнорують потреби користувачів з обмеженими можливостями. Проєкт вирішує проблему ізоляції таких користувачів, створюючи ігрову механіку, яка прощає моторні помилки та забезпечує мультисенсорний зворотний зв'язок.

Мета та основні завдання

  • Провести аналіз стандартів Web Content Accessibility Guidelines (WCAG) для мобільних додатків.
  • Розробити архітектуру нативного Android-додатку мовою Java.
  • Реалізувати ігрову механіку з оптимізованими хітбоксами (зменшення зони ураження на 30%).
  • Впровадити режим "Високого контрасту" для людей з порушеннями зору.
  • Додати тактильний відгук для інформування про ігрові події.

Методологія дослідження

Розробка здійснюється у середовищі Android Studio з використанням нативних інструментів Android SDK (Java, XML). Фізика гри реалізована програмно через розрахунок координат та використання об'єктів Rect для точної реєстрації зіткнень.

Очікувані результати

Інклюзивність

Повноцінна підтримка контрастних тем та вібраційного відгуку.

Швидкодія

Стабільні 50 кандрів на секунду (оновлення екрану кожні 20мс).

Готовий продукт

Робочий APK-файл гри, готовий до встановлення на Android-смартфони.
