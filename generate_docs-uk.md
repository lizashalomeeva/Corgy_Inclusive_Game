# Інструкція з генерації документації

## Огляд інструментів
Ми використовуємо **Javadoc** для коду та **Swagger UI** для API.

## Як згенерувати локально
1. Відкрийте термінал Android Studio.
2. Виконайте команду: `./gradlew generateJavadoc`
3. Згенеровані файли з'являться у папці `app/build/docs/javadoc/`.
4. Відкрийте `index.html` у браузері.

## Перевірка якості (Linting)
Увімкнено сувору перевірку `-Xdoclint:all`. Якщо у коді бракує тегів `@param`, `@return`, збірка завершиться помилкою.
