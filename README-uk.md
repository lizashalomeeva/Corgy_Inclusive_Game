# Corgy Inclusive Game

## Про проєкт
Це інклюзивна мобільна гра для Android, розроблена спеціально для користувачів із порушеннями зору та моторики. Проєкт використовує механізми високого контрасту та тактильного зворотного зв'язку для забезпечення комфортного ігрового процесу для всіх.

## Технології
* **Мова:** Java
* **Платформа:** Android SDK
* **Документація:** Javadoc, OpenAPI (Swagger)
* **Якість коду:** Android Lint, Git Hooks

## Правила документування коду 
У цьому проєкті документація розглядається як невіддільна частина коду. Якщо ви хочете долучитися до розробки, дотримуйтесь таких правил:

1. **Код:** Усі публічні класи, інтерфейси та методи повинні супроводжуватися вичерпними Javadoc-коментарями.
2. **Жива документація:** Для складних алгоритмів або бізнес-логіки обов'язково додавайте блок `<pre>` із прикладом використання коду з реальних JUnit тестів.
3. **Лінтер:** У проєкті налаштовано суворий контроль (`-Xdoclint:all`). Збірка не пройде, якщо ви забули описати параметри (`@param`) або значення, що повертається (`@return`).
4. **Багатомовність:** Будь ласка, синхронно оновлюйте як `README.md` (англійською), так і `README-uk.md` (українською).
